Code Quest: The PyTome of Knowledge is a mobile game-based learning application for Android devices designed to teach IT learners the fundamentals of programming using the Pythong programming language.

The project combines mobile accessibility with gamification to improve programming education. The game incorporates Python teachings into a 2D top-down role-playing adventure set in the fictional world of Binarius, blending storytelling with hands-on coding lessons and challenges.
